- CanvasLayer

## layout containers
- MarginContainer: box with margins
- VBoxContainer: vertical stack of boxes
- HBoxContainer: horizontal list of boxes

## adding custom fonts
- on label, under "Custom Fonts", click "[empty]", select "New DynamicFont"
- drag in font file
- set size
